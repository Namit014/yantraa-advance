**API Endpoint:** `/v1/activities/:projectName/subjectName`  
**HTTP Method:** `GET`  
**Purpose:** Retrieve a list of activity subject names filtered within the current project.  
**Requirements:** Accessible only for requests made within the context of the current project.  
**Success Response:** Returns a `200 OK` status.
