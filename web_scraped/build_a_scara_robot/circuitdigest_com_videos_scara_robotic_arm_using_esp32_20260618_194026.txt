# DIY SCARA Robot Based on RISC-V Architecture

## Overview

This is a DIY SCARA (Selective Compliance Assembly Robot Arm) robot designed for precision tasks such as assembly, sorting, and pick-and-place operations. The project combines affordability with high functionality for learning and small-scale automation.

## Core Processing

- **Microcontroller:** ESP32-C6 WROOM module (RISC-V architecture)
- **Connectivity:** Wi-Fi and Bluetooth capabilities
- **Native USB Interface:** USB Type-C port for computer communication

## Motion System

- **Stepper Motors:** NEMA 17 stepper motors
- **Stepper Drivers:** TMC2209 (4 units), known for smooth and efficient operation
- **Gripper Servos:** Two SG90S servo motors for precise object manipulation

## Power System

- **Primary Voltage Regulation:** LM2596-5 buck converter — steps down a wide input voltage to 5V
- **Secondary Regulation:** ADP7118AUJZ-3.3 — low-noise CMOS LDO from Analog Devices, regulates 5V down to 3.3V for the ESP32-C6
- **Supervisor IC:** Analog Devices MAX6899 — ensures reliable system boot-up
- **Visual Feedback:** Onboard RGB LED for status indications

## Software & Firmware

- **Control Software:** Written in Python, featuring direct control and record-and-playback mode for programming complex movements
- **Firmware Development:** Arduino IDE, enabling customization and further development
