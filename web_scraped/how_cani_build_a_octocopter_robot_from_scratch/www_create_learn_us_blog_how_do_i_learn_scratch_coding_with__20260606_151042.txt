Here are the extracted technical specifications, facts, numbers, and descriptions:

* **Scratch Coding Platform:**
	+ Free block-based coding platform for creating games, stories, and animations
	+ Designed for students aged 8-16, but can be used by everyone
* **Scratch Editor Interface:**
	+ Coding area: where you put your code (building blocks)
	+ Scratch coding block palette: left side of the screen with blocks
	+ Color-coded blocks:
		- Blue: Motion Controls movement
		- Purple: Looks Changes how a sprite looks
		- Yellow: Events Starts your code
		- Orange: Control Brain of your code
		- Teal: Sensing Eyes and ears of the sprite
		- Dark Orange: Variables Stores information like Score, Lives, or a Timer
	+ Scratch sprite and background pane: bottom right for managing characters (Sprites) and backgrounds (Backdrops)
	+ Scratch stage: top right stage is a playground with an invisible grid
* **Programming Overview:**
	+ Start a new project: click "Create" button on Scratch at MIT
	+ Code blocks: drag and drop blocks to the coding area
	+ Sprites: add or delete as many as you want, each time adding a sprite to the stage
* **Code Blocks:**
	+ Events blocks (yellow): starting blocks, must go at the top of any chunk of code
	+ Motion blocks (blue): allow your sprite to move, rotate, glide, or go directly to a specific position
	+ Looks blocks (purple): change the appearance of your sprite, e.g. size, color, costume, backdrops, and more
* **Examples:**
	+ Move 10 Steps (blue)
	+ Go to x: y: (blue)
	+ When Green Flag Clicked (yellow)
	+ Forever loops (orange)
	+ If...Then blocks (orange)
	+ Sensing blocks (teal)
	+ Variables (dark orange)
