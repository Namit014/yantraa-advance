* Building Your First Robot:
	+ Decide between programmable autonomous robot or remotely operated robot (ROV)
	+ ROV is a great way to learn mechanical aspects
	+ Fundamental aspect of a robot: motor
	+ Minimum number of motors: 2 (left wheel, right wheel)
* Motors:
	+ Should be used for skid-steering (tank-steering)
	+ 2.4 GHz R/C unit is recommended for ROV
* Programming:
	+ LEGO Mindstorms: a good introduction to programming, graphically oriented
	+ LEGO Education's self-balancing model "GyroBoy"
	+ Parallax Basic stamp or Arduino board for advanced electronics
* Tips:
	+ Don't just get a board, get a project book to guide you through electronics and programming
	+ Consider weight classes for combat/sumo robots
	+ Don't get frustrated, building robots is hard
