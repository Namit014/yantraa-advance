The Arduino IDE 2.3.10, a major release, is faster and includes a modern, responsive editor with autocompletion, code navigation, and a live debugger; it is open source and its source code is hosted on GitHub, while a nightly build offers a preview of upcoming features. The legacy Arduino IDE 1.8.19 is available for Windows 7 and newer. The Arduino Cloud Editor lets you code, upload, and access projects from any browser at no cost. Arduino App Lab 0.8.0 is a visual environment for the UNO Q board that combines prebuilt Bricks with AI models, supports both C++ sketches via the Arduino IDE and Python, and is open source under the GPL 3.0 license with source code available on GitHub. The Arduino Flasher CLI, also open source on GitHub, enables flashing Linux images onto the Uno Q (a Windows version is provided). Pre‑configured Linux images are offered for the Uno Q and Ventuno Q boards, delivering a ready‑to‑use software environment. The Arduino PLC IDE 1.1.0 allows programming with IEC 61131‑3 languages and mixing Arduino sketches, provides pre‑mapped resources, no‑code fieldbus support, and extensive debugging tools, and runs on Windows 10 (64‑bit) or newer. The Arduino CLI is a command‑line interface that integrates all IDE functionalities for building, compiling, uploading sketches, and managing libraries and boards. Arduino Lab for MicroPython is an experimental lightweight editor that supports board connection, code upload, file transfer, and an interactive REPL shell. The IoT Remote mobile app provides dashboard monitoring and control for Arduino Cloud, and Science Journal lets users collect sensor data from smartphones and Arduino‑connected devices.
