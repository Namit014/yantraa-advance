# SunFounder PiDog – Raspberry Pi Robot Dog Kit

## Overview

The PiDog is a feature-packed Raspberry Pi robot dog kit produced by SunFounder. Each limb is driven by its own individual servo, giving fine control over the robot's movement. The kit is recommended for those who want to get into robotics and programming, though the build process is time-consuming and laborious. Younger builders may need adult assistance during assembly, but the finished robot is simple to use and program.

## Kit Versions

- Kit with Raspberry Pi 4 included
- Kit without a Raspberry Pi (for those who already own one)
- Kit with Raspberry Pi Zero W 2 (lower-cost option for users who do not need the extra power of the Pi 4)

## Build Details

- **74 assembly steps** in the official instructions
- **Over 44 sets of parts and tools** included
- The kit includes its own screwdrivers and wrenches
- Extra screws, nuts, rivets, and bolts are bundled in the kit
- Parts are packed in three inner boxes; the numbers on the boxes are for shipping purposes only and do not correspond to assembly stages
- Component orientation is critical, especially for the legs and servo motors
- SunFounder provides small red stickers to help identify correct orientation of components

## Main Components

| Component | Function |
|---|---|
| **Camera Module** | Enables facial detection and object tracking; can be used to make the robot follow a user or react when a face is detected |
| **Ultrasonic Module** | Measures distance to objects in front of the robot |
| **Touch Switch Module** | Detects when the head of the PiDog is touched; used in SunFounder's Python examples to trigger reactions |
| **6 DOF IMU Module** | Accelerometer and gyroscope; can detect if the PiDog has fallen over or been picked up; assists with balance |
| **Sound Direction Sensor Module** | Uses an array of microphones to determine the direction a sound is coming from, allowing the PiDog to react accordingly |
| **ROBOT HAT** | The main control board; handles and controls all PiDog functionality, including all 12 servos |
| **RGB Board** | An array of controllable RGB lights; can display emotion/reaction or be used for debugging code |

## Servo Count

The PiDog uses **12 individual servos** (one per limb joint), all managed through the ROBOT HAT.

## Pre-Build Preparation

Before assembly, SunFounder recommends:

1. Install Raspberry Pi OS on the board
2. Follow SunFounder's instructions to install all required libraries
3. Enable SSH and Wi-Fi on the Raspberry Pi
4. This preparation simplifies the servo-zeroing process later

## Example Scripts / Functionality

SunFounder provides documented Python example scripts:

- **Wake-up script** – The PiDog stretches and then holds a sitting position until the script ends.
- **Patrol script** – The PiDog moves forward; if the ultrasonic sensor detects an object ahead, the robot stops, shifts to a standing position, and plays a barking sound.
- **Rest script** – The PiDog lies in a resting position; when the sound direction sensor detects a noise, the robot stands up.

Users can open and modify the example code to customize behavior.
