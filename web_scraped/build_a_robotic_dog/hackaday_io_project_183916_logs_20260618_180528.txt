# OpenCat - Open Source Quadruped Robotic Framework

**Developer:** Petoi
**Platform:** Arduino and Raspberry Pi-based
**Type:** Open-source quadruped robotic pet framework for Boston Dynamics-styled programmable robots

## Overview

OpenCat is an open-source framework for building programmable quadruped robotic pets, developed by Petoi. The framework supports the Bittle and Bittle X robot dog models.

## Key Features

- **Multi-language programming support:** Python, C#, Arduino, and block-based coding via Petoi Coding Blocks (a Scratch-like environment running on Mind+ with auto-translation to Python)
- **Hardware:** 8 joints and 8 legs for complex movement control
- **Sensors:** Supports various sensor connections (configuration varies between Bittle and Bittle X)
- **Bluetooth connectivity** for wireless communication
- **Pre-built and DIY assembly options** available; assembly typically takes 1–2 hours

## Community Projects

### ChatGPT Integration with Bittle
Data Scientist, Educator, and Roboticist Ömer Çolakoğlu from Turkey integrated ChatGPT with the Petoi Bittle robot dog, enabling:
- Real-time conversational interaction and instruction-following
- Voice chat via ChatGPT API
- Multi-language support (demonstrated in Turkish)
- Performance of tricks, dances (including Michael Jackson moves), handstands, waves, stretches, jumps, crawls, flips, and more
- Open-source code available on GitHub via Petoi's forum

Çolakoğlu also plans to enable two ChatGPT-powered Bittle robots to communicate and play with each other, conversing on topics such as the economy and sports. He is exploring further AI integration using Isaac Simulator.

## Educational Curriculum

### 4-Session Robotics Curriculum
Designed by Dr. Rongzhong Li (inventor of OpenCat, founder & CEO of Petoi), this curriculum was created as a short robotics course for university non-science majors but is adaptable for students of all ages. It is organized into four sections configurable as a one-week camp or taught weekly. The coding environment is Petoi Coding Blocks on Mind+.

### Semester-Long Block-Based Coding Curriculum
Petoi also offers a full semester block-based coding curriculum for beginners in quadruped robotics.

## Community

Petoi maintains a vibrant global community that regularly contributes AI, ML, and LLM integrations to enhance the capabilities of Petoi's robotic kits.
