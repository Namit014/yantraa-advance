# SunFounder PiDog Robot Dog Kit – Technical Overview

## Product Description

The **PiDog** by SunFounder is a robot dog kit designed for the Raspberry Pi platform. It features an **all-metal construction** with four legs, each with **three degrees of freedom** driven by servo motors, enabling a wide range of movements.

## Hardware Specifications

- **Sensors included:** Touch sensor, ultrasound sensor (obstacle detection, positioned as "eyes")
- **Servo motors:** 12 total (3 per leg × 4 legs)
- **Compute module:** Raspberry Pi Compute Module 4 (for AI/ML capabilities)
- **Battery:** Onboard LiPo battery with included charger
- **Battery life:** Approximately 30 minutes of wire-free operation
- **Compatibility:** Raspberry Pi 4, Pi Zero 2 W; reportedly compatible with Pi 5 (untested by reviewer)
- **Kit price:** $179.99 (Raspberry Pi **not included**)

## Predefined Actions & Capabilities

The kit ships with sample Python programs enabling the following behaviors:

- Marching, sitting, standing, panting, resting
- Howling, wagging tail, barking
- Head tilting
- Shaking hand
- Push-ups
- "Fly like Superman" (triggered when picked up)
- Face tracking
- Touch response (pat on head)
- Obstacle detection via ultrasound (barks at danger)
- Red ball tracking

## Software Stack

- **Language:** Python
- **Libraries:** OpenCV and other open-source projects
- **Control methods:** SSH into the Pi and run dedicated programs; SunFounder mobile app
- **Code:** Modifiable — users can write custom programs to teach new tricks

## Assembly

- **73 assembly steps** per the included instruction sheet
- All parts individually marked; screws sorted in labeled poly-bags
- Extra screws provided beyond required quantity
- Two screwdrivers included
- Official assembly videos: two parts, totaling over 50 minutes
- Reviewer completed hardware assembly and software installation across two evenings

## Known Issues (from reviewer's experience)

- Touch sensor can be damaged during assembly if pushed too hard in the wrong direction (repairable via soldering)
- One sample Python program initially did not respond to Ctrl+C to exit; this was subsequently fixed by SunFounder developers

## Community Support

SunFounder maintains an **active, well-moderated forum** for product users to discuss issues and seek help.

## Additional Notes

- The reviewer expressed interest in future voice-control integration (not currently available)
- SunFounder offers additional assembly kits for Raspberry Pi and Arduino platforms
