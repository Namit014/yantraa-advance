# Mike's Robot Dog – Technical Specifications and Build Instructions

## Overview

Mike Rigsby's project on Hackaday.io aims to replicate a Boston Dynamics Spot Mini robot dog at a fraction of the cost. While Spot Mini is commercially unavailable and a Chinese copy costs approximately $30,000, this DIY version can be built for **under $600 total** — a 99% cost reduction. A single four-leg module can be assembled in one week for **under $500**, with an additional **~$100** for power and control components (Arduino, battery, switches, voltmeter, wire, connectors).

## 2. Body Material Selection

The body requires approximately **200 square inches** of material. Several options were evaluated:

| Material | Weight (g/sq in) | Notes |
|---|---|---|
| 1/4" plywood | 2.15 | Too flexible; bending problems in early versions |
| **1/2" plywood** | **4.3** | **Selected — cut to shape and painted** |
| 1/4" acrylic | 4.0 | Difficult to work with |
| 5 mm foam board | 0.59 | Doubts about stiffness and secure connections |
| Carbon fiber sheets | — | Considered but deemed a separate project |

## 3. Leg Assembly Procedure

1. **Shoulder Servo:** Attach a servo motor to the "Super Servo Holder" using (4) M3 × 16 mm bolts, nuts, and flat washers. For the **left front leg**, set the shoulder servo to the CCW rotation maximum (**150°**). For the **right front leg**, set to CW maximum (**30°**). Left front and right rear legs share the same configuration; right front and left rear are mirrored.

2. **Axle Assembly:** Print (4) axles with shorter threads ("Axle Shorter Thread"). Attach servo horns. Slide through "Bearing Linearb." Thread "Fat Pieceb" onto this.

3. **Drilling:** Drill holes **25 mm deep** using a **7/64" drill bit**. Install (4) 20 mm M3 machine screws. Add the servo horn and back bearing piece.

4. **Servo Twist Motor:** Set to **90°**. Attach "Servo Twist Holder" to "Upper LegA" using (4) M3 × 12 mm bolts and nuts.

5. **Super Servo Combo:** Install motor using (4) M3 × 16 mm bolts, nuts, and flat washers. Attach to "Upper LegA" using (4) 3D-printed "Axle Bolts." Set motor to CW max (**30°**).

6. **Ankle Assembly:** Attach "Back Bearinga" to "Ankle Servo Conn" using (3) M3 × 8 mm bolts. Attach aluminum servo horn to "Ankle Servo Conn" using (2) 2-56 × 3/4" bolts, nuts, and lockwashers. Connect "Ankle Servo Conn" to "Lower Leg" using (1) "Axle Bolt." Connect "Foot Holdera" to "Lower Leg" using (2) "Axle Bolts."

7. **Foot:** Place "Axle" in "Foot" (loose fit). Insert foot/axle into "Foot Holdera." Secure using (2) "Axle Bolts."

8. **Final Leg Connection:** Connect lower leg assembly to upper leg assembly by attaching "Ankle Servo Conn" to the servo motor. Leg should be in sitting position. Insert and tighten servo horn screw. Adjust "Fat Pieceb" so the leg is in sitting position; insert and tighten servo horn screw.

9. **Mounting to Body:** Place "Servo Shoulder Holder" and "Bearing Linearb" on the plywood body. Assume sitting position. Secure using (4) #6 × 1-1/4" screws and (4) #6 × 3/4" screws. Insert servo twist motor into "Fat Pieceb." Install and tighten servo horn screw.

10. **Wiring:** Install 5-position terminal block using (2) #6 × 3/4" wood screws. Install wire guide using (1) #4 × 1/2" wood screw. Connect servo extender cable to "Super Servo Combo" servo motor. Route cable and secure with duct tape. Secure servo wire to "Servo Twist Holder" using a tie wrap. Secure wires in terminal blocks.

    **Terminal block pinout (outside looking in):**
    - Position 1 (leftmost): Upper servo control wire
    - Position 2: Servo twist control wire
    - Position 3: Lower (Super Servo Combo) control wire
    - Position 4: Positive
    - Position 5: Negative

    Repeat for all four legs.

## 4. Control and Power System

### Power Supply
- **8V sealed lead acid battery** with slide-on lugs (removable for recharging)
- Power supply mounted on **1/4" plywood**, attached to the body using **Velcro**
- A **disconnect plug** enables quick removal or swapping of the entire power supply
- **Digital voltmeter** for battery status monitoring (optional but recommended)
- **Two heavy duty toggle switches**

### Wiring
- Motor power terminal blocks connected using **#18 AWG wire**
- Arduino Uno provides control signals
- Connection order: Left front leg → Right front leg → Right rear leg → Left rear leg

### Head and Tail Construction
- **Head:** 8" diameter styrofoam ball (carved for shape); muzzle is a styrofoam oval; nose is a small circle; neck carved from styrofoam oval; ears from brown faux fur fabric; tongue from pink foam; finished with google eyes, paint, and glue. A 1" × 2" wood rectangle inserted into the neck mounts the head to the body.
- **Tail:** Brown giant chenille stem, doubled.

## 5. Walking Gait – Software Approach

The walking sequence follows an alternating diagonal leg pattern:

1. Lift the **left front** and **right rear** legs
2. Move the left front and right rear legs **forward and down**
3. Lift the **right front** and **left rear** legs
4. Return the left front and right rear legs to **normal position**
5. Move the right front and left rear legs **down to normal position**
6. **Repeat**

### Servo Pin Assignments and Default Positions

| Servo | Arduino Pin | Default Angle |
|---|---|---|
| Left Front Upper (lfupper) | 0 | 131° |
| Left Front Mid (lfmid) | 1 | 99° |
| Left Front Lower (lflower) | 2 | 42° |
| Right Front Upper (rfupper) | 3 | 46° |
| Right Front Mid (rfmid) | 4 | 98° |
| Right Front Lower (rflower) | 5 | 38° |
| Right Rear Upper (rrupper) | 6 | 150° |
| Right Rear Mid (rrmid) | 7 | 96° |
| Right Rear Lower (rrlower) | 8 | 30° |
| Left Rear Upper (lrupper) | 9 | 48° |
| Left Rear Mid (lrmid) | 10 | 102° |
| Left Rear Lower (lrlower) | 11 | 50° |

### Code Parameters
- `st` = number of steps
- `p` = delay value (**100 ms**)
- `tilt` = lean delay time (**200 ms**)

The code uses the Arduino `Servo.h` library. Each servo is attached to its respective pin in `setup()` and written to its default position. The gait loop alternates the diagonal leg pairs as described above.
