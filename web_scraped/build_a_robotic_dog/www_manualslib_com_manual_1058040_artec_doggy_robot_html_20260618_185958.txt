# Artec Doggy Robot — Assembly Instructions Manual

## Components List

- **Studuino Unit** (main controller board)
- **LED (red)**
- **Reflective IR Photo Reflector (Infrared Sensor)**
- **Basic Cube (white)** × multiple
- **Basic Cube (clear)** × multiple
- **Servomotor** × multiple
- **Triangle A (gray)** × multiple
- **Triangle A (clear)** × multiple
- **USB Cable**
- **Battery Box**
- **Half B** (structural component)
- **Sound Sensor**

---

## Pre-Assembly: Servomotor Calibration

Before building the robot, read **Section 6: Using Servomotors** in the **Studuino Icon Programming Environment Guide**, downloadable from:

> http://www.artec-kk.co.jp/robotist/

Building the robot without calibrating the servomotor may cause damage or improper functionality.

---

## Assembly Steps

### 1. Attaching Servomotors for the Legs
- Attach servomotors to the leg assemblies using the specified cubes and connectors.
- Ensure the Studuino unit is inserted correctly.
- Connect the servomotor cables to their corresponding ports on the Studuino unit, ensuring correct cable orientation.

### 2. Attaching Servomotor for the Head
- Attach the servomotor to the head assembly.
- Connect the servomotor cables to their corresponding ports on the Studuino unit, ensuring correct cable orientation.

### 3. Attaching the Reflective Infrared Sensor
- Connect the sensor cable: **Gray** wire aligns with the **circuit board side**; **Black** wire aligns with the **sensor side**.
- Connect the reflective infrared sensor cables to port **A5** on the Studuino unit.

### 4. Assembling the Head
- Assemble the head structure using Basic Cubes and Triangle A pieces.
- Connect the **LED (red)** cables to port **A4** on the Studuino unit.
- Ensure correct cable orientation (gray/black alignment).

### 5. Assembling the Torso
- Assemble the torso frame using Basic Cubes, Triangle A pieces, and Half B components in the sequence shown across the assembly steps.
- Connect the **Sound Sensor** cables to port **A6** on the Studuino unit.
- Ensure correct cable orientation (gray/black alignment).

### 6. Installing the Battery Box
- Mount the battery box onto the robot body.
- Connect the battery box cable to the **POWER** section on the Studuino unit.
- The battery box switch should be visible at the POWER location.

### 7. Replacing Batteries
- Use a **Phillips #1 screwdriver** to open the battery box.
- Replace batteries following the polarity markings (＋/−) inside the battery box.

---

## Completion Check

- Be cautious of cables that could become entangled in the moving parts of the motors, which may cause the robot to disconnect. Take care when arranging cables.
- Before operating, re-check the Assembly Instructions to confirm the robot has been assembled correctly.

---

## Operating the Doggy Robot

1. Install the Studuino Programming Environment software from:
   > http://www.artec-kk.co.jp/studuino/
2. Connect the **USB cable** from the PC to the Studuino unit.
3. Upload the desired program.
4. Turn the **battery box switch ON** — the robot will start walking.
5. **Immediately turn the switch OFF** if the robot does not begin walking as expected. Failure to do so may damage the servomotor.

---

## Sensor Calibration

Some sensors may not function properly after running the program for the first time. If sensors are malfunctioning:

- Click the **Submenu 1** tab to calibrate the sensors.
- A box will appear to adjust the **range settings** for each sensor.
