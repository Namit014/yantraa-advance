# SunFounder PiDog Robot Dog Kit – Technical Overview

## Product Description

The PiDog by SunFounder is a robot dog kit designed for the Raspberry Pi platform. It features an all-metal construction for durability and has four legs, each with three degrees of freedom powered by servo motors, enabling a wide range of movements. The kit includes an ultrasound sensor (positioned as the "eyes") for obstacle detection and a touch sensor for interaction (e.g., petting triggers tail wagging or barking).

## Compatibility

- Compatible with Raspberry Pi 4 and Raspberry Pi Zero 2 W
- Should work with Raspberry Pi 5 (untested by the reviewer)
- Uses a Raspberry Pi Compute Module 4 to enable AI and machine learning capabilities

## Pricing

- Kit costs **$179.99**
- **Raspberry Pi is NOT included** in the kit

## Hardware Assembly

- Assembly involves **73 steps** per the included instruction sheet
- All parts are properly marked and screws are packed in labeled poly-bags
- SunFounder provides extra screws beyond what is required
- Two screwdrivers are included in the kit
- Some screws are very small and can be challenging to handle
- Official assembly videos are available in two parts, totaling over 50 minutes

## Software

- SunFounder provides sample Python programs
- Based on Python, OpenCV, and other open-source projects
- Code is modifiable, allowing users to teach the robot new tricks
- Can be controlled via SSH or SunFounder's mobile app

## Preset Actions/Tricks

The PiDog can perform the following predefined actions:

- Marching
- Sitting, standing, panting, resting
- Howling, wagging tail, barking
- Tilting head
- Shaking hand
- Push-ups
- "Fly like Superman" (when picked up)
- Face tracking
- Responding to touch (pat on head)
- Danger detection via ultrasound (triggers barking)
- Red ball tracking

## Battery and Operation

- Includes an onboard LiPo battery with a charger
- Provides approximately **30 minutes** of wire-free operation

## Known Issues (from reviewer's experience)

- Touch sensor can be damaged if pushed too hard in the wrong direction during assembly; may require soldering to fix
- Some sample Python programs do not respond to Ctrl+C for exit, requiring the process to be killed explicitly
- No built-in voice control module (reviewer suggests this as a desired feature)

## Support

- SunFounder maintains an active, well-regulated forum where users can seek help and have moderators available for assistance

## Use Cases

- Suitable as a DIY project for hobbyists, makers, and tinkerers
- Can be a family activity with teenage children
- Applicable in educational settings (teachers can use it in labs to introduce students to robotics)
- Offers a learning curve that provides hands-on experience in robotics, Python programming, and embedded systems
