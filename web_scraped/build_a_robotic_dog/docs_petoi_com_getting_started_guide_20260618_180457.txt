# Getting Started Guide | Petoi Doc Center

## Overview

This guide covers setup and initial usage for Petoi robots, including Bittle, Bittle X, and Nybble.

## Assembly

- **Bittle** and **Bittle X** construction kits include build instructions.
- **Bittle X V2** (as indicated on the package barcode label): visit https://guide.petoi.com/
- **Nibble** has its own assembly instructions.

> **Note:** The robot is suitable for smooth surfaces. Movement on carpet may not be smooth.

## Recommended First Steps

1. Play with the mobile app
2. Play with the default actions
3. Play with simple joint control
4. Add more commands to the controller panel
5. Play with group commands to make the robot perform multiple actions in sequence
6. Play with voice commands (for Bittle X or any robot with the voice command module)

### Voice Command Troubleshooting

- If the robot doesn't respond to voice commands, see the troubleshooting guide.
- Voice command functionality can be turned off (e.g., in public spaces or quiet classroom settings) to avoid accidental triggers.

## Coding

- Follow the **Petoi Coding Blocks** curriculum to program robotics moves
- Visually design new robotics skills with **Petoi Skill Composer**
- Follow the **C++ curriculum** to code robotics moves in C++
- For Bittle/Bittle X, you may 3D-print a stand to keep the robot stationary while programming

## Competition Projects

Work on quadruped robotics competition projects.

> **Note:** All tutorials work on Bittle/Bittle X robot dogs and Nybble robot cat, even if one particular robot pet is used as an example.
