# AnyDesk 9.7.6 — Technical Specifications

**Developer:** AnyDesk
**License:** Freeware
**Supported OS:** Windows, macOS, Linux, Android
**File Size:** 7.9 MB
**Last Updated:** June 15, 2026

## Core Description

AnyDesk is remote desktop software that enables users to access their data, images, and applications from anywhere and share them with others. It delivers a fluent on-screen experience as if sitting directly in front of the remote device, supporting **60 fps** on local networks and most internet connections, and performs smoothly even with bandwidths as low as **100 kb/sec**. It is noted as a preferred solution for areas with poor internet connectivity.

## Codec & Performance

AnyDesk uses **DeskRT**, a proprietary video codec specifically designed for graphical user interfaces. Unlike conventional remote desktop applications based on outdated compression techniques (X11, RDP, VNC) or codecs designed for photos (JPEG) or video (H.264), DeskRT exploits GUI-specific image properties such as large areas of same color, high contrasts, sharp edges, repeating patterns, and linear translation of image contents to achieve high image quality and low response times.

## Encryption & Security

- **TLS 1.2** based encryption (the same standard used in online banking) for secure communication.
- **AES-128 bit encryption in GCM mode** is used by default for all connections.
- **AES-256** end-to-end data encryption is also referenced, ensuring transmitted data cannot be read by third parties, including AnyDesk itself.
- Both ends of a connection are cryptographically verified, preventing impersonation of an AnyDesk ID.
- Users can verify the encryption mode, verification status, and client fingerprint by hovering over the lock icon in the status bar.
- Licensed versions provide even stronger encryption.

## Access Control

- Remote access requires the user's **AnyDesk ID and password**; without these, unauthorized access is not possible.
- Additional options include **unattended access** and the ability to block specific devices.
- Remote sessions are typically **sandboxed**, meaning personal files remain inaccessible to the remote party unless explicitly shared.

## Installation & Portability

AnyDesk requires only **1 MB** of download, can be sent via email or run from a USB drive, and requires **no administrative privileges or installation**.

## Known Issues & Troubleshooting

- **Black screen / "Waiting for image":** Usually caused by the remote computer lacking a connected monitor. Connecting a display or using a dummy HDMI adapter resolves this.
- **Administrator privileges / UAC:** AnyDesk may prompt for elevated permissions for certain actions. Windows User Account Control (UAC) can interfere with these during remote sessions.
